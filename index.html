QUICK GUIDE

The Problem
I used to sit down with 20 physics mains questions, assign an hour, and somehow spend two when one question eats up 20 minutes, a "quick" break stretches to 30, and before you know it your session is off the tracks. This app keeps that from happening.

The Solution
You set a total time and a question count. The app divides the time equally and holds you to it, one question at a time. If you're doing 20 JEE Advanced physics PYQs in 3 hours, each question gets exactly 9 minutes. When the time's up for a particular question, it moves on.

The Buttons
Start: begins the session with the first question.
Done: you finished early. Any remaining time from this question gets added to your Saved Time bank, which you can spend later on a harder question.
Pause: you need more time on this question, or you're stepping away. The timer stops and you can see exactly how much extra time you're taking. Use this for bathroom breaks too.
Resume: picks up where you left off.

The Notes Section
I used to note question numbers I wanted to revisit once done with all questions. Use it however you like.

One Rule
This app is for a single focused session. Don't enter 600 minutes and 200 questions expecting to run it all day. Set it for one sitting, sit down, and start.
